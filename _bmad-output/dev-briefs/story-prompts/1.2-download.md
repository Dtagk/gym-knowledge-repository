Create yt_kg/download.py implementing Story 1.2: Throttled Audio Download.

Public API: one function `download()` with no arguments.

Implementation:
- Call init_db() from .db, query: SELECT * FROM videos WHERE downloaded_at IS NULL
- For each video row, download audio using yt-dlp Python API:
  - ydl_opts: format='bestaudio[abr<=96]/bestaudio', outtmpl='data/audio/%(id)s.%(ext)s',
    postprocessors=[{'key': 'FFmpegExtractAudio', 'preferredcodec': 'm4a'}],
    sleep_interval_requests=2, max_sleep_interval=5, quiet=True, no_warnings=True
  - Create data/audio/ directory before downloading
- On success: UPDATE videos SET downloaded_at=utcnow() WHERE video_id=?
- On any exception: UPDATE videos SET last_error=str(e), error_stage='download' WHERE video_id=?; log and continue
- Rows with downloaded_at IS NOT NULL are excluded by the WHERE clause (idempotent)

Imports: use utcnow and init_db from .db. Standard library only otherwise (no new deps).
Do not modify any existing files.

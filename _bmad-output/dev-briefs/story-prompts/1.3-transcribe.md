Create yt_kg/transcribe.py implementing Story 1.3: Local Transcription with faster-whisper.

Public API: one function `transcribe()` with no arguments.

Implementation:
- Call init_db(), query: SELECT * FROM videos WHERE downloaded_at IS NOT NULL AND transcribed_at IS NULL
- Load WhisperModel: try ("medium", device="cuda", compute_type="float16"); if CUDA unavailable
  fall back to ("small", device="cpu", compute_type="int8"). Detect via torch.cuda.is_available()
  or catch the faster_whisper CudaError.
- vad_filter=True is MANDATORY on every model.transcribe() call — never omit it.
- For each video:
  - audio_path = Path(f"data/audio/{video['video_id']}.m4a")
  - segments, info = model.transcribe(str(audio_path), vad_filter=True)
  - Write data/transcripts/{video_id}.json:
    {"language": info.language, "duration": info.duration,
     "segments": [{"start": s.start, "end": s.end, "text": s.text} for s in segments]}
  - On success: delete audio_path, UPDATE videos SET transcribed_at=utcnow()
  - On any exception: UPDATE videos SET last_error=str(e), error_stage='transcribe'; do NOT delete audio; continue

Create data/transcripts/ directory before writing.
Imports: faster_whisper.WhisperModel, json, pathlib.Path, logging, .db (init_db, utcnow).
Do not modify any existing files.

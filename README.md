# Gym Knowledge Repository

A personal project with two parts:

1. **Gym Knowledge Base** — a static web app with an interactive SVG body map as navigation. Click a muscle group to browse training, mobility, stretching, and recovery content for that area.

2. **YouTube → Knowledge Graph Pipeline** — a local pipeline that ingests gym/fitness YouTube channels, transcribes audio with Whisper, extracts a typed knowledge graph via a local LLM (Ollama), and exposes query tools to Claude Code via an MCP server.

Built as a learning project for faster-whisper, LanceDB, Kuzu, Ollama, instructor, and the MCP SDK.
